วิธีการใช้งาน
  -ใช้งานแบบปกติ(Manual) 
    - พิมพ์คำสั้ง npm i
    - พิมพ์คำสั้ง npm run dev
    - ให้เข้า path http://localhost:3000
  -ใช้งานผ่าน docker
    - พิมพ์คำสั้ง docker compose docker-compose.build.yml -d up 
    - พิมพ์คำสั้ง docker compose docker-compose.yml -d up app
    - ให้เข้า path http://localhost:3000
